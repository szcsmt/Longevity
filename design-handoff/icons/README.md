# Longevity Resort — Icons (SVG)

Every icon used on the website, as standalone vector SVGs (31 total).
All use `currentColor`, so they take the colour of their context (or set
`color` / `fill` in CSS). On the site they render gold (`#C9A96E`).

**Outline icons** (`stroke="currentColor"`), **solid** icons (`fill="currentColor"`).

## Library icons (Lucide, 19)
activity, arrow-up-right, briefcase, building-2, droplets, flame, gem,
heart-pulse, leaf, lightbulb, mic, moon, shield, sofa, sparkles, sun, trees,
waves, wind

## Custom UI icons (12)
arrow-right, chevron-left, chevron-right, chevron-down, close, cube-3d
(the "View it in 3D" mark), download, expand (the zoom/enlarge hint),
meditation (the "soundproof" figure), map-dot, map-pin (the estate pushpin),
whatsapp.
