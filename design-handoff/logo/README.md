# Longevity Resort — Logo (SVG)

Three vector versions, split from the master `LOGO.svg`. All are scalable
(viewBox only, no fixed size) and tightly cropped to their content.

| File | Contents |
|------|----------|
| `logo-lr.svg`   | The **LR mark** only (the circular emblem: LR monogram + waves) |
| `logo-text.svg` | The **wordmark** only (LONGEVITY RESORT) |
| `logo-full.svg` | The **complete logo** (mark + wordmark) |

Colour: gold `#f1c77e` (the original logo gold). To recolour, change the
`fill` on the `<path>`.
